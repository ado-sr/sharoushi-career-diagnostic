# 社労士キャリア診断ツール 📊

## 概要

社労士Lab公式のPR用キャリア診断アプリケーション。たった10問の質問で、ユーザーに最適な社労士キャリアパスを提案し、適切な社労士Labサービスへ誘導します。

## 🎯 目的

1. **リード獲得**: 潜在顧客との最初の接点を作る
2. **ブランディング**: 社労士Labの専門性をアピール
3. **バイラル拡散**: SNSシェアによる認知度向上
4. **サービス誘導**: 診断結果から各サービスへの自然な導線

## 📱 機能

### 診断機能
- 条件分岐型の質問フロー（7問）
- キャリア理論に基づく科学的診断
- 4つの診断結果タイプ:
  - **🌱 成長志向タイプ** → シャロスタへ誘導
  - **💼 実務重視タイプ** → 社労士Labコミュニティへ誘導
  - **⚖️ 専門特化タイプ** → リーガルラボへ誘導
  - **🚀 独立志向タイプ** → 社労士Labコミュニティへ誘導

### 学術的根拠
本診断ツールは以下の確立されたキャリア理論に基づいています:

1. **Edgar Schein のキャリア・アンカー理論**
   - MIT教授により開発された8つのキャリア価値観分類
   - 専門・職能別能力 (TF)、全般管理能力 (GM)、自律・独立 (AU)、保障・安定 (SE)、起業家的創造性 (EC)、奉仕・社会貢献 (SV)、純粋な挑戦 (CH)、生活様式 (LS)

2. **John Holland の RIASEC 理論**
   - 職業興味の6タイプ分類（現実的、研究的、芸術的、社会的、企業的、慣習的）
   - 個人の興味と職業環境の適合度を測定

各診断結果には、該当するキャリア・アンカーと職業興味タイプが明示され、科学的根拠に基づいたキャリア提案を行います。

## 🔧 バックエンドサーバーのセットアップ（X API連携用）

### X アカウント情報の自動取得機能について

Xアカウントのアイコンとニックネームを自動取得するには、Node.jsバックエンドサーバーが必要です。

### セットアップ手順

#### 1. サーバーディレクトリに移動

```bash
cd server
npm install
```

#### 2. X API キーの取得

1. [X Developer Portal](https://developer.twitter.com/) にアクセス
2. アプリケーションを作成
3. **Bearer Token** を取得（API v2用）
4. Free Tier でも使用可能（月間500,000リクエストまで）

#### 3. 環境変数の設定

```bash
cp .env.example .env
```

`.env` ファイルを編集:

```env
TWITTER_BEARER_TOKEN=your_bearer_token_here
PORT=3000
FRONTEND_URL=http://localhost:8080
```

#### 4. サーバーの起動

```bash
npm start
```

サーバーは `http://localhost:3000` で起動します。

詳細は `server/README.md` を参照してください。

### バックエンドなしで使用する場合

バックエンドサーバーを起動しない場合、Xアカウント情報は自動取得されませんが、ユーザーは手動でニックネームを入力できます。

## 📊 Googleスプレッドシート連携（診断データ収集）

診断結果を自動でGoogleスプレッドシートに記録できます。管理者が一覧表示で確認可能です。

### 記録される情報

- **基本情報**: 診断日時、ニックネーム、Xアカウント名
- **診断結果**: 診断タイプ、RIASECスコア、信頼度、レア度
- **セグメント**: 受験生、開業希望、転職検討中など（自動判定）
- **回答データ**: 24問の回答パス、経験レベル
- **アクセス情報**: User Agent、リファラー

### セットアップ（5分）

詳細は `google-apps-script/SETUP.md` を参照してください。

**簡単な手順:**

1. Googleスプレッドシートを新規作成
2. 「拡張機能」→「Apps Script」を開く
3. `google-apps-script/Code.gs` のコードをコピー&ペースト
4. ウェブアプリとして公開（アクセス権限：全員）
5. デプロイURLを `index.html` の `GOOGLE_SHEETS_URL` に貼り付け

### 集計機能

スプレッドシート上部のカスタムメニュー「📊 診断データ」から：

- **集計シートを作成**: セグメント別、診断タイプ別、経験レベル別の集計を自動生成
- **データフィルタ**: 受験生のみ、開業希望者のみなど絞り込み可能
- **Looker Studio連携**: 高度な可視化・ダッシュボード作成

### 取得できる分析

- 受験生 vs 開業希望者の割合
- 各セグメントのRIASEC特性の違い
- 流入経路別の診断数
- 日別・週別の診断数推移

### Google Analytics 設定

```html
<!-- index.html の </head> 前に追加 -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');

  // イベントトラッキング
  function trackEvent(action, label) {
    gtag('event', action, {
      'event_category': 'Career Diagnostic',
      'event_label': label
    });
  }
</script>
```

イベント設定例:
- `診断開始`: startQuiz()実行時
- `質問N回答`: 各質問の回答時
- `診断完了`: 結果表示時
- `SNSシェア`: シェアボタンクリック時
- `CTA クリック`: 各サービスへのリンククリック時
